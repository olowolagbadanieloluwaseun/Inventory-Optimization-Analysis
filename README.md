# 📊 Inventory Optimization Analysis Project (SQL + Dashboarding) 
**_Company: Nova Mart Supply Chain Ltd._**


## 📌 Project Overview
This project focuses on **Inventory Optimization Analysis**, aimed at deriving actionable insights from inventory data and solving key business challenges related to stock management and supply efficiency.

---

## 🔍 Business Objectives

- Which products are currently below their reorder level?
- Which category has the highest number of low-stock products?
- Which suppliers provide the most products?
- Are there overstocked products tying down capital?
- Which warehouse location holds the highest inventory volume?
- Which 5 products have the highest stock value but still require immediate restocking?
- What is the total stock value by category?

---

## 📈 Key Performance Indicators (KPIs)

- **Total Products**
- **Items Below Reorder Level**
- **Total Stock Value** *(Current Stock × Unit Price)*
- **Number of Categories**
- **Average Reorder Level**
- **Top Low-Stock Products**
